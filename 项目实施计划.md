# 内网资产清单网页工具实施计划

## 1. 项目目标

建设一个可在内网部署的网页工具，支持以下业务流程：

1. 用户登录系统后上传服务器资产总表。
2. 系统从数据库读取以下三类基础数据：
   - 项目-责任人映射表
   - 未配额主机列表
   - 暂不安装主机列表
3. 系统自动生成以下三个结果清单：
   - Agent在线未添加防护配置主机列表
   - Agent未安装主机列表
   - Agent防护中断主机列表
4. 页面支持结果预览、分别下载结果文件、历史记录查询与留存。
5. 基础数据支持后台维护，包括批量导入、搜索、新增、编辑、删除。

## 2. 需求范围

### 2.1 输入方式

1. 服务器资产总表由用户每次手动上传。
2. 其余三类基础数据通过系统后台管理并存储到数据库。
3. 总表页面支持点击上传和拖拽上传。

### 2.2 输出方式

1. 页面预览三个结果表。
2. 分别下载三个结果文件。
3. 保留历史生成记录，支持按日期、操作人、源文件名查询。

### 2.3 登录方式

1. 系统采用简单账号密码登录。
2. 未登录用户不可访问上传、生成、管理、历史页面。

## 3. 技术方案

### 3.1 技术栈

## 4. 数据模型规划

### 4.1 基础表

1. 用户表：保存账号、密码摘要、状态、创建时间。
2. 项目责任人映射表：保存企业项目、负责人、备注、更新时间。
3. 未配额主机表：保存服务器ID、IP地址、服务器名称、匹配键、备注。
4. 暂不安装主机表：保存服务器ID、IP地址、服务器名称、匹配键、备注。
5. 导入历史表：保存上传总表文件名、上传人、上传时间、处理状态。
6. 结果记录表：保存生成批次、操作人、时间、结果条数、结果文件路径、缺失负责人统计。

### 4.2 匹配规则

排除清单匹配键采用以下优先级：

1. 服务器ID
2. IP地址
3. 服务器名称

导入基础数据时，建议同步生成规范化匹配键，减少运行时重复计算。

## 6. 三个结果清单的生成规则

### 6.1 Agent在线未添加防护配置主机列表

1. 服务器名称
2. 服务器ID
3. IP地址
4. 配额ID
5. 服务器状态
6. Agent状态
7. 风险状态
8. 防护状态
9. 操作系统
10. 版本类型
11. 企业项目
12. 来源
13. 负责人

筛选规则：

1. 服务器状态 = 运行中
2. Agent状态 = 在线
3. 防护状态 = 未防护
4. 排除未配额主机列表
5. 按企业项目补充负责人

文件名称：

Agent在线未添加防护配置主机列表-YYYY-MM-DD

## 7. 页面功能规划

### 7.1 登录页

1. 账号密码登录
2. 登录失败提示
3. 登录后跳转工作台

### 7.2 工作台

1. 上传服务器资产总表
2. 展示上传结果和校验提示
3. 触发三个清单生成
4. 页面预览三个结果表
5. 提供分别下载入口
6. 展示负责人缺失项目提示

### 7.3 基础数据管理页

包含三个管理模块：

1. 项目-责任人映射表管理
2. 未配额主机列表管理
3. 暂不安装主机列表管理

每个模块支持：

1. 批量导入
2. 搜索
3. 分页
4. 新增
5. 编辑
6. 删除

### 7.4 历史记录页

1. 按日期查询
2. 按操作人查询
3. 按源文件名查询
4. 查看每次生成的结果条数
5. 重新下载历史结果文件

## 8. 实施阶段划分

### Phase 1：项目骨架与基础设施

1. 初始化 FastAPI 项目。
2. 建立配置、路由、模板、静态资源、数据目录。
3. 接入 SQLite。
4. 完成登录鉴权基础能力。

### Phase 2：基础数据后台管理

1. 实现三类基础数据的数据库模型。
2. 实现批量导入。
3. 实现搜索、新增、编辑、删除。

### Phase 3：清单生成能力

1. 实现总表上传与校验。
2. 实现字段映射与标准化。
3. 实现负责人补全逻辑。
4. 实现三个结果清单的生成。
5. 实现结果文件导出。

### Phase 4：页面完善与历史记录

1. 增加结果预览页面。
2. 增加历史记录查询。
3. 增加异常提示、空结果提示、缺失映射提示。

### Phase 5：部署与运维准备

1. 初始化管理员账号。
2. 编写部署说明。
3. 明确 SQLite 文件位置和备份方式。
4. 明确导出文件目录与清理策略。

## 9. 校验要求

1. 校验上传文件格式是否合法。
2. 校验总表是否包含必要列。
3. 校验三个清单的筛选条件是否正确生效。
4. 校验排除列表是否正确剔除对应主机。
5. 校验负责人是否按企业项目正确补充。
6. 校验结果文件命名是否符合日期规则。
7. 校验历史记录与导出文件是否可追溯。

## 10. 当前边界说明

本阶段不包含以下内容：

1. 对接外部资产平台自动拉取数据。
2. 单点登录。
3. 复杂权限体系。
4. 消息通知。
5. 高并发和分布式部署优化。
